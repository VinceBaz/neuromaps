brainnotation.stats.correlate_images¶

brainnotation.stats.correlate_images(src, trg, corrtype='pearsonr', ignore_zero=True, nulls=None)[source]¶

Correlates images src and trg

If src and trg represent data from multiple hemispheres the data are concatenated across hemispheres prior to correlation

Parameters

  • src (str or os.PathLike or nib.GiftiImage or niimg_like or tuple) – Images to be correlated

  • trg (str or os.PathLike or nib.GiftiImage or niimg_like or tuple) – Images to be correlated

  • corrtype ({'pearsonr', 'spearmanr'}, optional) – Type of correlation to perform. Default: ‘pearsonr’

  • ignore_zero (bool, optional) – Whether to perform correlations ignoring all zero values in src and trg data. Default: True

  • nulls (array_like, optional) – Null data for src

Returns

correlation – Correlation between src and trg

Return type

float
